What Is Blast3D?

Blast3D is an easy to learn, game development language with its Blitz3D origins. It is continued to be used by many game developers worldwide, in a large part due to the continually active developer community.


3D Everywhere

Whether you want to create action packed 3D games like First Person Shooters, Survival, Adventures, RPGs, whatever... Blast3D can do it! Blast3D makes programming so easy that anyone can now produce games of great complexity.

2D Games

Games that rely on physics, AI, multiplayer games or with many features. You should also be able to create them and with a little more time and resources you can also make RPGs, Tower Defense, Adventure Games, Roguelikes, Fighting games.

Easy To Learn

The overall syntax, design and set of features might trick you into believing Blast3D is pretty rudimentary. However, with the right combination of commands you'll be able to create just about anything, both in 2D and 3D environments.
